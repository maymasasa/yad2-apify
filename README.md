# Yad2 Cars Scraper

An Apify actor that scrapes car listings from [Yad2](https://www.yad2.co.il) — Israel's largest classifieds platform.

See [`.actor/README.md`](.actor/README.md) for full documentation including input parameters, output schema, usage examples, and deployment instructions.

## Quick Start

```bash
npm install
npx apify run
```

Results are stored in `storage/datasets/default/`.
